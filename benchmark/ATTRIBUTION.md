# GeoGen Benchmark — Bluebonnet Learning Dataset

This folder contains benchmark data derived from the **Bluebonnet Learning** geometry textbooks
(grades 6, 7, and 8). Bluebonnet Learning is a State of Texas Open Educational Resource (OER):
the materials are owned by the State of Texas / Texas Education Agency and made available free for
use, modification, and sharing under a Creative Commons license. The grades 6–8 mathematics
content was adapted, with permission, from materials originally created by Carnegie Learning, Inc.,
and aligned to the Texas Essential Knowledge and Skills (TEKS).

## How the Benchmark Was Created

Activities and problems were extracted from Bluebonnet Learning geometry units using the `marker`
tool. Each extracted problem was filtered to geometry-relevant content and used to generate
student-facing diagram prompts. See the top-level project README for full methodology details.
